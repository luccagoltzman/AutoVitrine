# AutoVitrine - Catálogo Premium de Estética Automotiva

Um catálogo moderno e interativo para serviços de estética automotiva, desenvolvido com foco em experiência mobile-first e funcionalidades avançadas.

## 🚀 Funcionalidades Principais

### 📱 Design Mobile-First
- Interface responsiva otimizada para dispositivos móveis
- Navegação intuitiva com menu hambúrguer
- Carregamento rápido e experiência fluida
- PWA (Progressive Web App) com instalação nativa

### 🛠️ Simulador de Pacotes Personalizados
- **Montagem de Pacotes**: Clientes podem combinar serviços e ver descontos automáticos
- **Estimativa de Preços**: Sistema inteligente que calcula preços baseado no tipo de veículo
- **Descontos Progressivos**: 
  - 1 serviço: 10% de desconto
  - 2 serviços: 20% de desconto
  - 3+ serviços: 30% de desconto
- **Multiplicadores por Veículo**:
  - Hatch: 1.0x
  - Sedan: 1.2x
  - SUV: 1.5x
  - Pickup: 1.8x

### 🖼️ Galeria Comparativa Antes/Depois
- Slider interativo para comparação de trabalhos
- Filtros por categoria de serviço
- Detalhes completos de cada projeto
- Visualização otimizada para mobile

### ⭐ Sistema de Avaliações
- Avaliações reais com fotos dos trabalhos
- Sistema de pontuação visual (estrelas)
- Estatísticas de satisfação
- Depoimentos com informações do cliente e veículo

### 🎯 Programa de Fidelidade Digital
- **Cartão Fidelidade**: Sistema de pontos e recompensas
- **Benefícios**:
  - A cada 5 lavagens, a 6ª é grátis
  - 10% de desconto em serviços premium
  - Prioridade no agendamento
  - Ofertas exclusivas via WhatsApp
- Interface visual do cartão com progresso

### 📅 Sistema de Agendamento Integrado
- Agendamento direto via WhatsApp
- Integração com calendário
- Confirmação automática
- Lembretes de agendamento

### 💡 Área de Dicas de Manutenção
- Dicas práticas para manter o veículo
- Tags organizadas por categoria
- Conteúdo educativo e engajante

## 🚀 Como Usar

### Para Clientes
1. **Navegue pelos serviços** usando as categorias
2. **Monte seu pacote** na seção "Pacotes"
3. **Selecione seu tipo de veículo** para cálculo preciso
4. **Veja os descontos** aplicados automaticamente
5. **Agende via WhatsApp** com um clique

### Para Desenvolvedores
1. Clone o repositório
2. Abra `index.html` em um servidor local
3. Para PWA completo, sirva via HTTPS
4. Personalize cores e textos conforme necessário
